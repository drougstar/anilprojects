# anilprojects

This is my LIVE portfolio website, served at https://anilprojects.com via GitHub Pages.

Rules for Claude:

- The owner is a beginner — explain simply, avoid jargon, comment tricky code.
- This is a static site: index.html (structure), style.css (looks), script.js (behavior). No build step, no Node.
- IMPORTANT: pushing to the `main` branch publishes straight to the live internet site. Always verify changes in the local preview first, and ask the owner before pushing.
- Local preview: use the dev server in .claude/launch.json (Python's built-in web server on port 8000).
- The four sub-projects (week, guess, dice, random-color) live in their own repos under github.com/drougstar and have their own subdomains.
